/getCountries (GET, HEAD)

Query Params

Name (optional): name or part of a name to filter by

Region: region of countries to filter by

Returns

A json array of country names. Format: {string:{countryName:string}}

Examples

Request: /getCountries?name=&region=None

Result:{Country 1:{countryName:name}, Country 2:{countryName:name}}

Request: /getCountries?name=Al&region=Europe

Result:{Aland Islands:{countryName:Aland Islands}, Albania:{countryName:Albania}}

In the client, all displayed countries are links, and clicking on them shows the JSON data for that country

/getSubregions (GET, HEAD)

Query Params

Name (optional): name or part of a name to filter by

Region: region of countries to filter by

Returns

A json array of country subregions, the idea being to get the subregions of a country or region. Format: {string:{countrySubregion:string}}

Examples

Request: /getCountries?name=&region=None

Result:{Country 1:{countrySubregion:subregion}, Country 2:{countrySubregion:subregion}}

Request: /getCountries?name=Al&region=Europe

Result:{Aland Islands:{countrySubregion:Northern Europe}, Albania:{countrSubregion:Southern Europe}}

/getTimezones (GET, HEAD)

Query Params

Name (optional): name or part of a name to filter by

Region: region of countries to filter by

Returns

A json array of timezones, the idea being to get the timezones of a country or region. Format: {string:{abbr:string, name: string, gmtOffset: int}}

Examples

Request: /getCountries?name=&region=None

Result:{AFT:{abbr:AFT, name: Afghanistan Time, gmtOffset: 16200}}

Request: /getCountries?name=United States&region=None

Result:{HST:{abbr:HST, name: Hawaii-Aleutian Standard Time, gmtOffset:-36000}}

/getCapitals (GET, HEAD)

Query Params

Name (optional): name or part of a name to filter by

Region: region of countries to filter by

Returns

A json array of countries and their capitals Format: {string:{countryName:string, capital: string}}

Examples

Request: /getCountries?name=&region=None

Result:{name:{countryName:name, capital: capital}}

Request: /getCountries?name=United States&region=None

Result:{United States:{countryName: United States, capital: Washington}}

/addCountry (POST)

Body Params

Name: name of the new country

Region: region of the new country

Returns

a message if a new country was created, Format: {message:string}

Nothing is returned if an existing country name is input (country region can be updated)

Examples

Request: /addCountry

Body: {name: New Country, region=Asia }

Result:{message: Created Successfully}

Request: /addCountry

Body: {name: New Country, region=Polar}

Result:None (region of New Country is updated)

/addData(POST)

Body Params

Name: name of the country to add data to

Data: name of the data being added

Value: value of the data being added

Returns

a message if a there was a bad request, Format: {message:string}

Nothing is returned if valid data was added (new data can be checked from /getCountries)

Examples

Request: /addData

Body: {name: United States, data:Size, value:really big }

Result:None (client displays that it has been updated)

Request: /addData

Body: {name: United States, data: value: }

Result:{message: Invalid data or value, id: badInput}
